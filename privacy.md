# プライバシーポリシー

最終更新日：2026年6月27日

「SUNARU」（以下「本アプリ」）は、ユーザーのプライバシーを尊重し、以下の方針に従って個人情報を取り扱います。

## 1. 取得する情報

### 1.1 アカウント情報
- メールアドレス（メールでの新規登録時）
- Google アカウントのメールアドレスおよび表示名（Google ログイン利用時）
- パスワードはハッシュ化された状態のみ保持され、当方が平文で閲覧することはありません

### 1.2 旅程データ
本アプリ内でユーザーが作成・登録した以下のデータ：
- 旅の名前、開始日、終了日、通貨設定、タイムゾーン設定
- スポット情報（名前、種類、時刻、費用、メモ、Google マップ URL、緯度経度）
- 配置情報（どの日のどの余白ゾーンに候補を入れたか）

### 1.3 位置情報
- 「現在地」ボタンを押した場合に限り、端末の位置情報（緯度経度）を取得します
- 取得した位置情報は地図表示のためにのみ使用し、当方サーバーへ送信・保存することはありません

### 1.4 端末情報・利用ログ
- アプリの動作に必要なログ（エラーログ等）
- 個人を特定する目的では使用しません

## 2. 利用目的

取得した情報は以下の目的のためにのみ利用します：
- 本アプリの機能提供（旅程の保存・同期・閲覧）
- 複数端末間でのデータ同期
- 共有リンク機能による旅程の閲覧専用共有
- 不具合の調査・改善

## 3. 第三者提供

当方は、ユーザーの同意がある場合または法令に基づく場合を除き、取得した情報を第三者に提供しません。

ただし、本アプリは以下の外部サービスを利用しており、それらのサービスの仕様上必要な範囲で情報がやり取りされます：

| サービス | 提供元 | 目的 | 送信される情報 |
|---|---|---|---|
| Supabase | Supabase Inc. | データベース・認証 | アカウント情報、旅程データ |
| Google Maps SDK for Android | Google LLC | 地図表示 | 端末情報、地図閲覧時のリクエスト |
| Google Places API | Google LLC | 場所検索 | 検索キーワード |
| Google OAuth | Google LLC | ログイン | Google アカウント情報 |

各サービスのプライバシーポリシーは各社のウェブサイトをご確認ください。

## 4. データの保管

- 旅程データおよびアカウント情報は、Supabase（AWS 東京リージョン）のデータベースに暗号化されて保管されます
- 端末内（SQLite）にも同データのコピーが保存されます

## 5. データの削除

ユーザーは以下の方法で自身のデータを削除できます：
- アプリ内で旅を削除すると、Supabase 上のデータも論理削除されます
- ログイン画面の「アカウントを削除」から、自身のすべての旅・スポット・配置データを一括で削除できます
- ログイン情報（メールアドレス）の記録自体の完全な削除を希望される場合は、お問い合わせ窓口までご連絡ください

## 6. 共有リンク

ユーザーが共有リンクを発行した場合、リンクを受け取った他ユーザーは該当する旅程データを閲覧専用で参照できます。リンクの発行・停止はユーザーが任意で行えます。
